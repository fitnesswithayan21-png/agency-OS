# Development Roadmap

## Phase 0 - Foundation (this branch)
Docs, full SQL migration (schema, indexes, RLS, triggers, seeds), Next.js scaffold, Module 1 Auth, dashboard + portal shells, CI.

## Phase 1 - Sales Engine
Leads pipeline board + scoring; CRM profile + timeline; Proposals (builder, client approval, PDF); Contracts (versions, e-signature, expiry).

## Phase 2 - Delivery Engine
Projects (Kanban dnd-kit, timeline, tasks, milestones, deliverables); Team (directory, invites, workload, time tracking); Files (storage browser, previews).

## Phase 3 - Money + Communication
Invoicing (builder, recurring, payments, finance dashboard); Communication (realtime chat, activity feed, notifications); Portal completion.

## Phase 4 - Intelligence + Admin
Analytics (revenue, profit, conversion, completion, utilization, retention); Automations UI; Admin panel (users, roles, security, audit log).

## Phase 5 - Hardening
Playwright E2E, rate limiting, monitoring, accessibility, performance.
